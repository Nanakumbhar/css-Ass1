What is css?

  1. CSS stands for Cascading style sheets.

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.

  3. Css is not neccesary CSS saves a lot of work.It can control the layout of multiple web pages all at once

  4. External style sheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements 1
  • Internal - by using a style element in the head section.
  • External - by using a link element to link to an external CSS file. however, in this tutorial we will use inline and internal stules , because this is easier todemonstrate ,and easier for you to try it yourself .

When somparising CSS class vs ID ,this difference is that CSS class applies a style to multple elements .ID on the other hand , apploes a style to one unique element .ID is also special in that you can use a special URL to link directly to an element and it is used in JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page.Once an elements has been targeted, a style or set of styles can be applied to the element.

There is wide range of selectors avalable.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
